## Design direction (1 paragraph)
"Fish With A Local," taken literally: a workboat-honest, dawn-to-dark site where the signature section is a time-of-day rail (First Light / Full Sun / Golden Hour / After Dark) mapping every real service Brian offers to the hour it happens. Condensed uppercase type reads like a tide chart, coral is reserved for action, and every photo is Brian's own.
